# Variables
在rust中，变量是默认不可变的
当变量开始绑定一个值时，就不能进行变更
可以使用mut关键字让变量成为可变变量
In Rust, variables are immutable by default.
When a variable is immutable, once a value is bound to a name, you can’t change that value.
You can make them mutable by adding mut in front of the variable name.
